Scanning Practices

When a scan request comes in each item must be evaluated to determine which scanner should be used for digitization. Factors for deciding this include the size of the item, the requested resolution and the fragility of the item.

If the item is 12" by 17" or less and can be placed face down without undue harm then it should be scanned on the flatbed scanner. Film or negatives of any type will also be scanned on the flatbed. Use of the flatbed gives us more control over the resolution at which the item is scanned, the output size of the files and lighting.

If the item is too large to fit the flatbed or it is a book that would be harmed by placing it face down on the scanner, then it should be digitized on the planetary scanner.

Microtek Scanmaker 1000XL (Flatbed)
Items should be placed facing down with the top of the item toward the front of the scanner if possible. To access the scanning software open Photoshop, go to the file menu, to Import and click on "Microtek ScanWizard Pro." All items should be scanned in "RGB Colors" mode at 600ppi or the resolution requested, whichever is higher. In the upper right hand of the ScanWizard there is a small icon of an image. Clicking that will show the options for scanning Negative Film, Positive Film and Reflective. Everything that is not Negative or Positive film should be scanned as "Reflective." Click the "Overview" button to preview the item being scanned. Adjust the borders of the red and white box so that they surround the area to be scanned. Finally click "Scan." When the area is scanned an image of it will be imported into Photoshop. Close out of the scan wizard by clicking on the "X" in the upper right corner. The image in Photoshop can then be rotated and cropped, but no other adjustments should be made. The resulting file should then be saved to the appropriate collection folder on the R Drive as a TIFF with no compression. Should the patron request something other than a TIFF, or a resolution lower than 600 or a grayscale image then the necessary adjustments may be made and the resulting file saved to the appropriate patron scans folder on the R Drive. If the patron requests a full color, 600ppi (or higher) TIFF then the original file can also be saved or copied to the appropriate patron scans folder on the R Drive.

At times, a request will specify that the item should be scanned at a percentage of its size or with an output of a certain width or height. In the left window of the scanning wizard the percentage may be adjusted to achieve these results. Never scan an item at less than 100%! When scanning an item at a certain height or width the percentage should be adjusted so that those dimensions read slightly higher than the requested output. This gives room for cropping, so that the resultant file will still be at or greater than the requested output. Be sure to check the image size in Photoshop (Image menu, Image Size) before saving to be sure that the resulting file fits the requested specs.

i2S Digibook Suprascan A0 (Planetary)
Items too large or too fragile for the flatbed scanner may be scanned with the planetary scanner. The book cradle should be used for books and the tabletop for flat items such as maps and artwork. The light and color balance should be set each day that the planetary scanner is used to make sure that the most even light distribution is achieved. Maximum resolution is determined by the size of the item being scanned. The taller it is, the lower the resolution at which we can capture it. This means that for many larger items we will not be able to achieve scans at our standard 600ppi. Items should be scanned at 600ppi or the highest possible resolution. Often flat items can be turned sideways on the tabletop so that they are shorter and thus able to be scanned at a higher resolution. These images can later be rotated in Photoshop.

File Folder Structure
Images should always be saved in two locations, the Collection Folder and the Patron Request folder.

Collections
Always save to the Collection folder first, as this is where we will keep our highest resolution, full color TIFFs. The collection folders can be found at:
R:\Digital Services\IMAGES\- Collections

There are then subfolders for the following collections: LLMVC, Manuscripts, McIlhenny, Rare Books and University Archives.

For items that belong to the LLMVC, McIlhenny or Rare Books collections, new folders should be created under the appropriate heading for each book scanned. The folder name should contain the Call Number and title.

For example, the folder for the Elk's Souvenir at CallNo. F 379 B33E44 c2 would be:
R:\Digital Services\IMAGES\- Collections\LLMVC\F 379 B33E44 c2 - Elks Souvenir

The folder for The Dapuri drawings by Henry J. Noltie at CallNo. QK98.2 .N64 2002 OVER would be:
R:\Digital Services\IMAGES\- Collections\McIlhenny\QK98 2 N64 2002 OVER - the Dapuri Drawings

Manuscript item folders should be named slightly differently, with the Mss. Number, and the title of the collection.

For Example, the folder for the Turnbull (Dudley and Family) Papers, MssNo. 2907 would be:
R:\Digital Services\IMAGES\- Collections\Manuscripts\2907 - Turnbull (Dudley and Family) Papers

Folders for University Archives should be named with the records group and the records group name

For Example, the folder for the Office of the President is:

R:\DigitalServices\IMAGES\- Collections\University Archives\A0001 - Office of the President Records\

 

After saving an image to the proper collection folder, that image (or its derivative) should be saved in the Patron Requests folder

Patron Requests
If a patron requests a 600ppi (or higher), full color TIFF, the the image can be copied over to the folder for that patron's order. If they request a lower resolution, grayscale or a JPG, then that modified file should be saved to their patron requests folder. Be careful when creating derivative files not to save over the original!

Patron request folders can be found at:
R:\Digital Services\IMAGES\- Patron Requests

Folders should be created for each order a patron submits, named with the date it was signed, the patron's name and the order number, such as:
R:\Digital Services\IMAGES\- Patron Requests\YYYYMMDD - Doe, John - 1234-A


When an order has been scanned, the images saved to the Collection folder should be made "read only" so that changes made to create derivative files will not overwrite the original scans.
